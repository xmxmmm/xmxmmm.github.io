zsh和oh-my-zsh的安装与配置

正式安装之前，在终端输入以下命令，查看系统现有的shell：

cat /etc/shells

在终端直接输入以下命令，进行安装zsh：

apt install zsh

遇到点问题，APT 包管理器在下载 zsh 相关依赖时，被强制从 HTTP 重定向到了一个异常地址（含 VLAN 和 MAC 信息），系统禁止此类不安全重定向，导致下载失败。

APT 依赖本地的「软件源列表」（/etc/apt/sources.list 及相关文件）获取包地址，若列表过时或缓存异常，可能导致地址跳转错误。首先执行更新：

  • 更新软件源列表
    apt update

更换软件源为国内镜像
